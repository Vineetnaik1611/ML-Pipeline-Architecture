Welcome to the Student Exam Performance Predictor

This is a Machine Learning web application that predicts students' exam scores in mathematics based on their demographic information, parental education, lunch type, test preparation course, and prior reading & writing scores.

The application includes a complete ML pipeline:

  • Data Ingestion & Cleaning
  • Feature Transformation & Preprocessing
  • Model Training & Selection (Random Forest, XGBoost, Gradient Boosting, etc.)
  • Prediction via a web interface
